# Neirah Ho - Personal Portfolio Website

A responsive personal portfolio website built using vanilla HTML, CSS, and JavaScript to showcase projects, skills, and experience. The site features smooth scrolling, scroll-triggered animations, and a fully functional serverless contact form.

---

## Features

- **Smooth scroll navigation** — Custom cubic-bezier easing function for polished anchor link transitions
- **Scroll-triggered animations** — Entrance animations for experience cards, skill pills, and the contact form powered by the Intersection Observer API
- **Contact form** — Serverless form handling via the Formspree REST API with async/await, dynamic error handling, and real-time success/error UI feedback
- **Responsive layout** — Flexbox-based design with adaptive breakpoints for mobile and desktop
 
---

## Tech Stack

- HTML5
- CSS3 (Flexbox, keyframe animations, CSS transitions)
- JavaScript (ES6+, Fetch API, Intersection Observer API)
- [Formspree](https://formspree.io) — serverless form backend
- [Google Fonts — Quicksand](https://fonts.google.com/specimen/Quicksand)
